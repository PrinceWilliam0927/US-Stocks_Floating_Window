# US-Stocks_Floating_Window
US Stocks Floating Window
This is a floating U.S. stock quote window script written in Python.

It opens a small desktop window that lets you monitor U.S. stock prices in near real time.

Main features:

Displays a watchlist of U.S. stocks
Allows adding/removing ticker symbols, such as AAPL, NVDA, TSLA
Fetches quote data from Yahoo Finance
Supports regular market, pre-market, after-hours, and overnight prices
Allows refresh interval settings
Supports window transparency
Supports always-on-top mode
Provides price movement alerts
Saves settings in us_stock_float_config.json
